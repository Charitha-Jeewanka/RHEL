###### Default shell : GNU - bash
# SHELL BASICS
Commands have 3 parts
1. Command to run
2. Options 
  	- To adjust behaviour of command
3. Arguments
	- Typically targets of the command

###### example
`usermod -L user01`
1. Command = `usermod`
2. Option = `-L`
3. Argument = `user01`
